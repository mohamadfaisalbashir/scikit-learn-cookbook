# scikit-learn Cookbook: Summary Notes

<a style="width: 400px" href="https://www.amazon.com/scikit-learn-Cookbook-recipes-machine-learning/dp/1836644450"><img alt="scikit-learn Cookbook Cover" src="./cover.png" style="width: 400px; height: auto; padding: 10px;"></a>


**Original Authors:** Julian Avila & Trent Hauck

This repository contains a collection of Jupyter Notebooks summarizing practical recipes and core concepts from the book **"scikit-learn Cookbook"**. These notes are structured to provide hands-on solutions to common machine learning problems using Python and the `scikit-learn` library, covering everything from data preprocessing and model building to evaluation and production deployment.

## 📂 Repository Structure

The summaries are divided into 13 main modules that follow the learning path of the repository:

* **`01_Common_Conventions_and_API_Elements_of_scikit_learn.ipynb`**
    An introduction to the scikit-learn ecosystem, focusing on the standard estimator API, design principles, and general coding conventions.
* **`02_Pre_Model_Workflow_and_Data_Preprocessing.ipynb`**
    Techniques for preparing your data before modeling, including scaling, encoding categorical variables, and imputing missing values.
* **`03_Dimensionality_Reduction_Techniques.ipynb`**
    Methods to reduce the feature space for better performance and visualization, utilizing algorithms like PCA (Principal Component Analysis).
* **`04_Building_Models_with_Distance_Metrics_and_KNN.ipynb`**
    Implementing k-Nearest Neighbors and exploring how different distance metrics impact model predictions.
* **`05_Linear_Models_and_Regularization.ipynb`**
    Applying Linear Regression along with regularization techniques (Ridge, Lasso, ElasticNet) to prevent overfitting on continuous target variables.
* **`06_Advanced_Logistic_Regression_and_Extensions.ipynb`**
    A deep dive into logistic regression for classification tasks, including hyperparameter tuning and handling complex boundaries.
* **`07_Support_Vector_Machines_and_Kernel_Methods.ipynb`**
    Utilizing SVMs and the kernel trick to map data into higher dimensions for solving complex non-linear classification problems.
* **`08_Tree_Based_Algorithms_and_Ensemble_Methods.ipynb`**
    Building robust predictive models using Decision Trees, Random Forests, and Gradient Boosting techniques.
* **`09_Text_Processing_and_Multiclass_Classification.ipynb`**
    Handling natural language data (NLP), vectorizing text, and extending binary classifiers to handle multiple classes.
* **`10_Clustering_Techniques.ipynb`**
    Exploring Unsupervised Learning approaches to group similar data points using algorithms like k-Means and DBSCAN.
* **`11_Novelty_and_Outlier_Detection.ipynb`**
    Techniques for identifying anomalies and outliers in datasets using Isolation Forests, One-Class SVMs, and other robust algorithms.
* **`12_Cross_Validation_and_Model_Evaluation_Techniques.ipynb`**
    Advanced methods for assessing model reliability, implementing rigorous cross-validation strategies, and interpreting evaluation metrics.
* **`13_Deploying_scikit_learn_Models_in_Production.ipynb`**
    Best practices and recipes for serializing, exporting, and serving trained scikit-learn models in real-world production environments.

## 🛠️ System Requirements

To run the `.ipynb` files in this repository, it is recommended to set up a Python environment with the following libraries:
* `scikit-learn`
* `numpy`
* `pandas`
* `matplotlib`
* `scipy`
* `jupyter`
